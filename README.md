# SauceDemo Automation Framework – Cypress

## Project Overview

This project demonstrates the design and implementation of a reusable test automation framework using Cypress and the Page Object Model (POM).

The framework validates an end-to-end checkout workflow within the SauceDemo sample web application while emphasizing maintainability, scalability, and clean test data management.

This project was created to showcase automation skills that support hybrid QA roles requiring both manual testing and automated validation.

---

## Framework Highlights

- Page Object Model (POM) design pattern  
- Custom Cypress commands  
- Fixture-based test data management  
- Modular and maintainable folder structure  
- Continuous Integration workflow using GitHub Actions  
- End-to-end workflow validation  

---

## Automated Test Coverage

The automation framework validates:

- User authentication  
- Product selection  
- Add-to-cart functionality  
- Cart verification  
- Checkout process  
- Order confirmation
